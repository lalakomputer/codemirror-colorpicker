# Image

Color 를 알아가다보니 결국은 Image 로 와 버렸습니다. Image 는 Color의 모음이죠.

Color 가 할 수 있는 일들은 이미지에서도 할 수 있고 이미지만의 톡득한 상태를 만들어 낼 수 있습니다.

그래서 색을 다룰 수 있다는 것은 곧 이미지를 다룰 수 있다는 말과 같다.

이미지를 다룰 수 있는 몇가지 중에 먼저 필터에 대해서 알아보자.

# Filter
